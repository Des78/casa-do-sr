The configuration of the IFTTT triggers and devices/events is done via an excel file in dropbox.

This application requires an authorization code from the target Dropbox account.

The Excel config file will be stored in a private dropbox Apps folder, the application won't have access to any personal folder on the dropbox account.

1. Sign in with your Dropbox account to get the authorization code


2. Enter the authorization code here
